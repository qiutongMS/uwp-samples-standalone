# UWP Samples Standalone

40 UWP samples extracted from [Windows-universal-samples](https://github.com/microsoft/Windows-universal-samples) for migration analysis.

These samples are referenced by Microsoft Learn documentation pages in the /windows/apps/develop/ docset.

## Structure

Each sample retains its original structure from the source repo:
- `Samples/<SampleName>/` — individual sample projects
- `SharedContent/` — shared resources used by multiple samples

## Purpose

This repo is used to:
1. Verify each sample builds as UWP
2. Analyze migration feasibility to WinUI 3 / Windows App SDK
